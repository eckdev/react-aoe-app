## Run Locally

Clone the project

```bash
  git clone https://github.com/eckdev/react-aoe-app.git
```

Go to the project directory

```bash
  cd react-aoe-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
